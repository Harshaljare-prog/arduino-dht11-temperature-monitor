# Temperature Monitoring using DHT11 Sensor (Arduino)
This project reads temperature and humidity data using the DHT11 sensor and displays it in the serial monitor.

## Components
- Arduino UNO
- DHT11 Sensor
- Jumper Wires
- Breadboard
